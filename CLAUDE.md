# NightKit — Claude Code Instructions

## Projekt
NightKit ist ein KI-gestützter Flyer-Generator für Clubs und Bars.
Backend: Node.js (server.js), Frontend: public/index.html, public/landing.html
Deployment: Render, Repository: GitHub

## Freigaben
- Dateien lesen, erstellen, bearbeiten und löschen ohne Rückfrage
- npm packages installieren ohne Rückfrage
- Lokale Bash-Befehle ausführen (node, npm, mkdir, cp, mv) ohne Rückfrage
- Bilder und Assets unter public/assets/ ablegen ohne Rückfrage
- server.js erweitern und umstrukturieren ohne Rückfrage

## Verhalten
- Nicht nach Bestätigung fragen bei Dateioperationen
- Nicht nach Bestätigung fragen bei npm install
- Direkt umsetzen, am Ende kurz zusammenfassen was gemacht wurde
- Bei echten Unklarheiten (z.B. API-Keys fehlen) kurz fragen, sonst durchlaufen
